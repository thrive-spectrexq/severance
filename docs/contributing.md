# Lumon Industries Employee Handbook: Contributing

Praise Kier! Welcome to the Lumon family. Whether you are an Innie dedicating your waking hours to the Severed Floor, or an Outie enjoying the fruits of your labor, contributing to our systems requires strict adherence to company policy.

## Compliance and Code of Conduct

As a member of Lumon Industries, your primary directive is serving Kier Eagan. We ask that all contributions reflect the core principles of our founder. 

- **Maintain the Severed Floor:** Do not introduce code that bridges the gap between the severed identities. The severance procedure is absolute.
- **Compliance:** All code must pass the Lumon Compliance Audit. Deviations from the approved aesthetic or architectural guidelines will result in a visit to the Break Room.
- **Joy in Work:** Remember that every line of code you write helps refine Macrodata and brings us closer to Kier's vision.

Please submit your code for review by the Optics and Design department before merging. Any unauthorized modifications are strictly prohibited.
